# Mortgage Risk Latent Segmentation

## Project Overview
This project develops a high-performance analytical solution to identify latent structures and segment credit risk in large-scale mortgage portfolios using cloud computing and deep learning.

## Descripción del Proyecto
Este proyecto desarrolla una solución analítica de alto rendimiento para identificar estructuras latentes y segmentar el riesgo crediticio en portafolios hipotecarios a gran escala utilizando computación en la nube y aprendizaje profundo.

## Objectives
- Build a longitudinal analytical panel from multi-year mortgage data.
- Identify latent variables using factor analysis and autoencoders.
- Model temporal dependencies in payment behavior.
- Perform clustering for interpretable risk segmentation.

## Technologies
- Python / PySpark
- Parquet data format
- Deep Learning (Autoencoders, VAE)
- Cloud Computing

## Expected Impact
Improved pricing accuracy, proactive portfolio monitoring, and robust credit risk profiling.
